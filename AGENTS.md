# AI Agent Instructions for `trip`

## 项目概述

这是一个静态旅游网站项目，基于 HTML、CSS 和 JavaScript。项目目标是展示旅游目的地、行程推荐、景点详情、攻略与联系方式。

## 关键文件

- `README.md`：项目目标、功能需求、页面结构和技术选型说明。
- `index.html`：网站首页。
- `destinations.html`：目的地展示页面。
- `itineraries.html`：行程推荐页面。
- `attractions.html`：景点列表与搜索页面。
- `attraction-detail.html`：景点详情页面。
- `guides.html`：旅游攻略页面。
- `about.html`：关于我们页面。
- `styles.css`：全局样式。
- `scripts.js`：导航菜单与景点搜索交互逻辑。
- `images/`：本地 SVG 占位图资源。

## 开发注意事项

- 这是一个纯静态网站，没有后端或构建工具。
- 以简单、可维护、响应式设计为主，避免引入复杂框架。
- 优先使用本地资源，避免依赖外部占位图片服务。
- 任何新增页面应保持导航一致并在导航栏中加入链接。
- 搜索、详情页、表单类功能应使用轻量前端交互实现。

## 代理工作建议

- 优先阅读 `README.md` 以理解用户需求和页面结构。
- 在修改页面时，保持页面样式统一、结构清晰。
- 仅在必要时添加新文件，避免创建过多重复页面。
- 代码更改应兼容移动端与桌面端。
